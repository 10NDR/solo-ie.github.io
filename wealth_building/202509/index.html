総資産の算入範囲#

本レポートで示す「総資産額」に含めるもの・含めないものは以下の通りです。

含むもの

  • 現金・預金（普通 / 当座 / MRF）
  • 投資信託（NISA 枠含む）
  • 株式（特定 / 一般 / NISA 枠内の国内外株式 / ETF）
  • NISA 評価額（投信・株式と二重計上しない整理で集計）
  • iDeCo 時価評価額
  • 小規模企業共済（掛金累計ベース / 月末評価）

含まないもの

  • 個人年金などの保険（解約返戻金も含めず）
  • 退職金見込み額
  • FX 口座残高
  • ポイント / マイル / ポイ活残高
  • 不動産資産（自宅）
  • 住宅ローン負債（純資産ではなくアセットサマリーのため）
