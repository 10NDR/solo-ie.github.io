管理人について

このブログは、独身アラフォーが運営しています。

基本情報

  • 年齢: アラフォー

  • 職業: 零細企業の会社員、自称フルスタックエンジニア（Java, Python, JavaScript, TypeScript, React, DB, AWS, Azure など）

  • 居住地: 福岡県（現在は福岡県でフルリモート勤務しつつ、山口県で家づくり中）

  • 趣味: 自転車（ロードバイク）、お酒（あまり強くない）、資産運用

経歴

  • 山口県で高校まで過ごす
  • 大学進学を機に他県で一人暮らしを開始
  • 就職を機に東京へ行き、10 年間勤務
  • 転職を機に福岡県でフルリモート勤務を開始
  • 現在は山口県で一条工務店のアイキューブによる注文住宅を計画中

このブログを始めた理由

注文住宅について調べていると、結婚されているご家庭向けの情報がほとんどです。

コミュニケーションを重視した間取りや、お子様が喜ぶ間取りなど、家庭向けの内容が目立ちます。

独身だからこそできる家づくりや、独身ならではの悩み・工夫などを発信し、私のような人間もいるんだと、少しでも似た境遇の方の後押しになればと思いました。

家づくりのコンセプト

  • 将来の変化に対応できる間取り
  • 趣味を楽しめる空間づくり
  • メンテナンスしやすい設計
  • 独身ならではのリセールを気にしない自由な間取り
  • 親の介護も見据えた立地選び

一条工務店ご紹介制度

一条工務店をご検討の方、ご紹介制度というものがあります。
ご紹介する側・される側どちらにも特典があり、私自身もとても助かっています。
以下のような特典がありますので、「一条工務店の展示場に行ってみようかな」という方は、ぜひお気軽にお声がけください。（展示場来場後でも大丈夫です）
ご紹介の際はお互いに個人情報が必要となりますので、私も自分の情報を開示します。
不要な連絡を避けたい場合は、資料と紹介状だけを取り寄せる形も可能です。
連絡先からお気軽にご連絡ください。

ご紹介特典

ご紹介される側（あなた様）

  • 展示場で QUO カードプレゼント
  • カップボードなど住宅設備の中から 1 つプレゼント（数十万円分の特典です！）
  • 優先して優秀な営業マンの担当付

管理人の特典

  • i-ポイント 6,000P（一条工務店のポイントで 1P=1 円相当）
  • ご紹介した人が契約された場合は 100,000P

大きいですね。ブログや YouTube でご紹介制度について触れている方は多いですが、紹介する側のメリットを書かない人も多いです。
個人的に、紹介する側のメリットも正直に書いておきたいと思い、しっかり明記しています。
